## Film Stockpot 0.3.4

Windows x64 installer for grading **NegPy flat 16-bit TIFF** exports with film-stock presets and Fuji Frontier–style operator controls.

### Download

- **FilmStockPot_x64_0.3.4.exe** — full installer (recommended)
- Requires Windows 10/11 x64

---

## Highlights

### Spectral crosstalk correction (NegPy-compatible)

- Per-stock **3×3 crosstalk matrices** embedded in preset JSON (13 color stocks), derived from NegPy calibration data
- **Crosstalk slider** (0.00–1.00) under the film-stock dropdown: **0 = off**, **0.50 = default**, **1.00 = full matrix**
- Runs in the pipeline **after base auto-levels and before neutralize**, so dye-layer unmixing works on the scan’s native channel balance
- Setting is saved in sidecars and applied on export / batch CLI
- Synchronous preview updates when adjusting crosstalk (no stale async renders)

### Datasheet-driven preset refresh

- New **datasheet extraction and refresh workflow** to update `FilmPresets/` from manufacturer PDF data
- Pipeline stages driven by datasheet sensitometry:
  - **Per-channel tone curves** and color crossover
  - **Acutance** (MTF-derived micro-contrast)
  - **Reciprocity failure** compensation in shadows
  - **PGI/RMS grain** metadata and grain-extraction tuning
- Dev scripts: `extract_datasheets.py`, `refresh_presets_from_datasheets.py`
- Refreshed Kodak Portra, Ektar, Ultramax, Fuji consumer/pro stocks, and more

### Preset and pipeline improvements

- **`resolve_preset_data()`** — rendering always uses the latest installed preset JSON (e.g. crosstalk matrices stay current)
- Improved **headroom limiting** on crosstalk so channel decreases are not over-attenuated on bright flat scans
- Updated preset metadata, tone curves, and confidence blocks from datasheet research

## Existing features (unchanged)

- Film-stock library (Kodak, Fujifilm, Ilford, HARMAN, and more)
- Frontier-style density, gamma, CMY, tone, highlight/shadow, saturation, sharpness
- Live RGB histogram (linear / log)
- Pipeline preview stages and split compare
- Non-destructive JSON sidecars per image
- Single and batch 16-bit TIFF export with naming templates
- Headless CLI for automation
- Self-contained sidecars (portable recipes)

---

## NegPy handoff

1. Process and export a **flat 16-bit TIFF** from NegPy  
2. Open the folder in Film Stockpot  
3. Pick a film stock, tune Frontier controls (and crosstalk if desired)  
4. Export graded 16-bit TIFFs — originals are never modified

## Upgrade notes

- **0.3.4 installer (re-uploaded again):** fixes startup crash (`QtWidgets` DLL load failed) by pinning matching PyQt6/Qt6 wheels, bootstrapping Qt6 DLL search paths at launch, and smoke-testing every build. **Uninstall any previous 0.3.4 install first**, then download the latest `FilmStockPot_x64_0.3.4.exe`.

- Sidecars store `adjustments.crosstalk` as **0.00–1.00** (older 0–100 or NegPy-style 1.0–2.0 values are migrated automatically)
- After installing, **presets ship inside the app folder** — edit `FilmPresets/` beside the executable or use the repo JSON during development
- Selecting a new film stock resets crosstalk to **0.50** (default); your slider value is restored from sidecars when reopening an edited image
