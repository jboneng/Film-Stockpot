## Film Stockpot 0.4.5

Windows x64 installer with a faster, more accurate **Grading** tab — color wheels, L/R/G/B curves, and live preview improvements.

### Download

- **FilmStockPot_x64_0.4.5.exe** — full installer (recommended)
- Requires Windows 10/11 x64

---

## Highlights

### Grading tab — color wheels

- Shadow / midtone / highlight **color wheels** with per-zone luminance sliders, blending, and balance
- Fixed wheel colors matching the preview (no more red/green inversion)
- Stronger, more visible grading response
- **Real-time preview** while dragging: scanner adjustments are cached separately from grading, with optional **GPU acceleration** (View menu) for instant wheel updates
- Fixed stale-preview bug where grading-only changes could fail to appear until another control moved

### Grading tab — curves

- **L / R / G / B curve editor** with draggable control points
- Click the curve to add a point; right-click a point to remove it (endpoints stay fixed)
- Smooth **Bezier** curves between points (not straight-line segments)
- Faint **luminance histogram** drawn behind the curve for reference while you edit

### Windows installer reliability

- PyQt6 / Qt6 versions pinned to matching releases
- Runtime bootstrap so frozen builds find Qt DLLs on a clean Windows install
- Every build is smoke-tested before publish

---

## Existing features (unchanged)

- Film-stock library (Kodak, Fujifilm, Ilford, HARMAN, and more)
- Spectral crosstalk correction with per-stock matrices and 0–1 slider
- Frontier-style density, gamma, CMY, tone, highlight/shadow, saturation, sharpness
- Live RGB histogram (linear / log)
- Pipeline preview stages and split compare
- Non-destructive JSON sidecars per image
- Single and batch 16-bit TIFF export with naming templates
- Headless CLI for automation
